# NullFlow Project Page

This folder is a static GitHub Pages website for:

**NullFlow: One-Step Generative Reconstruction**

## Files

- `index.html` — main webpage
- `style.css` — webpage styling
- `assets/NullFlow_Preprint.pdf` — linked paper PDF
- `assets/nullflow_subspace.png` — method illustration extracted from the paper
- `assets/extracted_p5_0.png` — visual reconstruction comparison
- `assets/extracted_p6_0.png` — sample-averaging plot

## Deploy with GitHub Pages

1. Create a new GitHub repository, for example `nullflow`.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then click **Save**.
6. The page will be available at `https://<username>.github.io/<repo>/`.

For a CIG organization page, create or upload this folder to the desired repository under `wustl-cig`, then enable GitHub Pages the same way.
